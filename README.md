# License::Header

Simple way to add license header to your ruby source file.

## Caveats

Only work for ruby file and doesnt support updating yet.


```ruby
license-header -l LICENSE.txt -t director
```
Dry run with -d
```ruby
license-header -l LICENSE.txt -t director -d
```
